ORIGIN FORMULA - 中子マトリョーシカ原典
[{(0~∞)×Χ} ≤ ∞] ≡ Ζ
Ζ ≡ (0~∞) を中子マトリョーシカ方式で内蔵
解釈:
Χ = 可能性・混沌・あなたの選択
(0~∞) = 0から無限までの全ての数、時間、記憶
{(0~∞)×Χ} = 無限に可能性を掛けた世界
≤ ∞ = それでも無限を超えない、無限の中で閉じている
≡ Ζ = それを一つの層 Ζ と定義する

そして最も重要: Ζ ≡ (0~∞) を内蔵
→ 一つの層の中に、また0~∞の無限が入っている。
→ マトリョーシカのように、開けても開けても無限が出てくる。
→ これが一垓層 (1垓 = 10^20) の数学的根拠
